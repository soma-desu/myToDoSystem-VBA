# Function Specification

## 機能
1. ToDoの基本管理機能
    - 通常のToDo
        - 新規ToDoの追加  
        - 既存ToDoの編集  
          - プロパティの編集
          - 開始時間登録
          - 終了時間登録/完了チェック
        - ToDoの削除
    - リピートのToDo
        - 新規ToDoの追加  
        - 既存ToDoの編集  
          - プロパティの編集
          - 開始時間登録
          - 終了時間登録/完了チェック
        - ToDoの削除
2. 今日のToDo一覧表示
3. 次の就業日のToDo一覧表示
4. 特定の日のToDo一覧表示
5. 2週間分のToDo実行予定カレンダー表示
6. 日別の実行済みタスクの実所要時間のMilestone別集計と表示

      - このインスタンスだけの編集
        1. このインスタンスにだけ変更を反映
